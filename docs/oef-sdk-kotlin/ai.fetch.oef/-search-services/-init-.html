SearchServices(searchId: Int, query: Query)

This message is used for searching agents which provides the queried service. This search will only search on the local node the agent connected to, and not the whole OEF network.

Parameters

searchId - identifies the search query. This id will be used by the sender in order to distinguish different incoming search results.

query - i.e. a list of constraints defined over a data model.
