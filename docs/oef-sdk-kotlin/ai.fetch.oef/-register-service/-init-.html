RegisterService(msgId: Int, serviceDescription: Description, uri: OEFURI = OEFURI())

This message is used for registering a new agent in the Service Directory of an OEF Node. The service agent is described by a Description message object.

Parameters

msgId - identifier of the message

serviceDescription - the description of the service provided by the agent

uri - resource identifier for the service. Using this field one agent is able to register multiple service.
