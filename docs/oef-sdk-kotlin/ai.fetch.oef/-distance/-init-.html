Distance(center: Location, distance: Double)

Class that implements the 'distance' constraint type. That is, the locations we are looking for must be within a given distance from a given location. The distance is interpreted as a radius from a center.

Examples: Define a location of interest, e.g. the Tour Eiffel val tour_eiffel = Location(48.8581064, 2.29447) 

     Find all the locations close to the Tour Eiffel within 1 km
     ```
     close_to_tour_eiffel = Distance(tour_eiffel, 1.0)
     ```

Parameters

center - the center from where compute the distancee

distance - the maximum distance from the center, in m
