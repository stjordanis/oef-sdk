Description() Description(attributeValues: List<KeyValue>, dataModel: DataModel? = null, dataModelName: String = dataModel?.name ?: "")

Description of either a service or an agent so it can be understood by the OEF and other agents. Contains values of the description, and an optional schema for checking format of values. Whenever the description is changed (including when it is create), the attribute values will checked to make sure they do not violate the attribute schema.

Example:

    val It = descriptionOf(
                 descriptionPair("title", "It"),
                 descriptionPair("author", "Stephen King"),
                 descriptionPair("genre",  "horror"),
                 descriptionPair("year", 1986),
                 descriptionPair("average_rating", 4.5),
                 descriptionPair("ISBN", "0-670-81302-8"),
                 descriptionPair("ebook_available", true)
             )
