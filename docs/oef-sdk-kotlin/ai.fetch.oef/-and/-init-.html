And(constraints: List<ConstraintExpr>)

A constraint type that allows you to specify a conjunction of constraints. That is, the And constraint is satisfied whenever all the constraints that constitute the and are satisfied.

Example: All the books whose title is between 'I' and 'J' (alphanumeric order) but not equal to 'It'

val c = And(listOf(Constraint("title", Range.StringPair(("I", "J"))), Constraint("title", Relation.NOTEQ("It"))))
