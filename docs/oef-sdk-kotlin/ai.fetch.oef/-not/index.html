Not

class Not : ConstraintExpr

A constraint type that allows you to specify a negation of a constraint. That is, the Not constraint is satisfied whenever the constraint that constitutes the Not expression is not satisfied.

Example: All the books whose genre is science fiction, but the year is not between 1990 and 2000

val c = And(listOf(Constraint("genre", Relation.EQ("science-fiction")), Not(Constraint("year", Range.IntPair((1990, 2000))))))
