IN(values: List<T>)

Class that implements the 'in set' constraint type. That is, the value of attribute over which the constraint is defined must be in the set of values provided.

Examples: All the books whose genre is one of the following: Horror, Science fiction, Non-fiction val c = Constraint("genre", Set.In(listOf("horror", "science fiction", "non-fiction")))
