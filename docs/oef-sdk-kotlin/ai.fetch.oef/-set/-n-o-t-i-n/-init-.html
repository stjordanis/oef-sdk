NOTIN(values: List<T>)

Class that implements the 'not in set' constraint type. That is, the value of attribute over which the constraint is defined must be not in the set of values provided.

Examples: All the books that have not been published neither in 1990, nor in 1995, nor in 2000 val c = Constraint("year", Set.NOTIN(listOf(1990, 1995, 2000)))
