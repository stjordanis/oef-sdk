CFP(messageId: Int, dialogueId: Int, destination: String, targetId: Int, query: CFPQuery, context: Context = Context())

This message is used to send a Call For Proposals.

Parameters

messageId - unique identifier for a message, given dialogue

dialogueId - identifies the dialogue in which the message is sent

destination - the public key of the recipient of the message

query - describes the resources the sender is interested in

targetId - the identifier of the message to whom this message is targeting, in a given dialogue

context - Context object used to send the message to the agent's specific role (the target agent has multiple services registered, and the source agent is trying to communicate regarding one of the services)
